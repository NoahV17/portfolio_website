Visual statistics - ML

Welcome to my final project for CSC 310 Programming for Data Science.

The goal of this site is to help show the relationship between two columns in a dataset.
We will visualize data using a scatter plot, then to classify the data's relatioship with a regression model projected on top of the scatter plot.


To use the web app, please choose two different varibales to compare, and view the outputted graph and its calculated regression line.

  • What do you notice?
  • Is there a relatioship between the x and y variables?
If not, try other categories that might have a relationship.
