History

2017 marks the inception of my GitHub account, when I built and published my first website from scratch in a high school web development class.
I was instantly hooked after being taught how to publish to our own unique github domain. And being able to push any changes to it through github, then seeing the changes become present in real time was mind blowing.
Now I am a part of a college class teaching Git, GitHub, and system implementation, and I figured it was the perfect time to rebuild my site, even if for the third or fourth time at this point.
