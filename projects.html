Galactic Defender



Built using Python's Pygame library, "Glactic Defender" employs object-oriented programming principles to manage game entities efficiently. The game features a diverse array of classes, including Background, Ship, Asteroid, Coin, and Mothership, each contributing to the overall gameplay experience. Collision detection, sprite animation, and user input handling are seamlessly integrated to create a responsive and immersive gaming environment.

Key Features: Dynamic sprites, multiple levels, boss fights, score tracking, radomized elements, and progressive level difficulty
