Galactic Defender


Built using Python's Pygame library, Glactic Defender employs object-oriented programming principles, as well as manual memory allocation to manage an ongoing flow of game entities. The game features 3 levels, 3 difficulty options, and a diverse array of classes, including Background, Ship, Asteroid, Coin, and Mothership, each contributing to the overall gameplay experience. Collision detection, sprite animation, and user input handling are seamlessly integrated to create a responsive and immersive gaming environment.

Key Features: Dynamic sprites, multiple levels, boss fights, score tracking, radomized elements, and progressive level difficulty

Task Tracker

Mar 2024 - May 2024

A task tracking app that helps all organize their events in a calendar style format. With the help of data structures and algorithms such as the ternary search, self-sorting array, and comb sort, and a front-end built in unity, our team was able to put together a highly functional application that runs on C++ and C# to enhance your daily life.

Employing the data structures and implementing them into a handful of helper functions allowed us to incorporate many sorting and tracking methods that help display the app's results on the Unity interface.

The above image shows the newly installed app without local data.
The next two lines add tasks through the terminal using the compiled C++ program.

./task-track ADD 091324 Work "Complete Project X" 10./task-track ADD 090524 Home "Clean the dishes" 5./task-track ADD 091324 Home "Return Amazon order" 7

After adding the tasks, the user can view them in the Unity interface, or lookup any tasks through the command line.

The folowing ONDAY command looks up and parses any tasks that are due on the date passes into the command, in the example shown it is for.

This is used by the front-end to help lookup the details for when you click on a date that you want to see the given tasks for. If you click on the 13th of September, 2024, it will run the command, the command will output all tasks on that day to the temp.txt file, and the app will read and format the text for the temp file for viewing in the front-end.

./task-track ONDAY 091324
